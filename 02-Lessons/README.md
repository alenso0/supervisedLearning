# Lessons — Structured Data Science & ML Curriculum

This is the full instructor-provided curriculum, in teaching order. Each lesson folder has slides (`.pptx`) and/or a worked notebook (`.ipynb`). If you're new to data science or ML, **start here, in order** — `02-Lessons` builds up the foundations before `03-Practice` lets you drill the same ideas hands-on.

| # | Lesson | What you'll learn |
|---|--------|--------------------|
| 01 | [Introduction to Data Science](Lesson_01_Introduction_to_Data_Science/) | What data science is, the typical workflow |
| 02 | [Working With Pandas](Lesson_02_Working_With_Pandas/) | DataFrames, dates/time deltas, text data |
| 03 | [Data Visualization (Matplotlib & Seaborn)](<Lesson_03_Data_Visualization(Matplotlib and Seaborn)/>) | Plotting basics through advanced Seaborn/Plotly |
| 04 | [Math and Statistics Fundamentals](<Lesson_04_Math and Statistics Fundamentals/>) | Linear algebra and statistics needed for ML |
| 05 | [Probability Distribution](Lesson_05_Probability_Distribution/) | Probability theory and common distributions |
| 06 | [Data Wrangling](Lesson_06_Data_Wrangling/) | Cleaning and reshaping messy data |
| 07 | [Feature Engineering](Lesson_07_Feature_Engineering/) | Creating and selecting useful features |
| 08 | [Introduction to Machine Learning](Lesson_08_Introduction_to_Machine_Learning/) | Core ML concepts and terminology |
| 09 | [Supervised Learning: Regression](Lesson_09_Supervised_Learning_Regression_and_its_Application/) | Linear/polynomial/ridge regression, RANSAC |
| 10 | [Supervised Learning: Classification](Lesson_10_Supervised_Learning_Classification_and_its_Application/) | Logistic regression, decision trees, k-NN, SVM, Naive Bayes |
| 11 | [Ensemble Learning](Lesson_11_Ensemble_Learning/) | Bagging, boosting, voting, stacking |
| 12 | [Time Series Algorithms](Lesson_12_TimeSeries_Algorithms/) | Forecasting with ARIMA/SARIMA |
| 13 | [Model Deployment](Lesson_13_Model_deployment/) | Packaging and serving a trained model (Streamlit, etc.) |

## Where to go next

Once you've read a lesson, jump to the matching topic in [`../03-Practice/`](../03-Practice/) and run the practical notebook yourself.
